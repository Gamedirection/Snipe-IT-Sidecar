# Snipe-IT Sidecar

Operational playbooks for running Snipe-IT beyond inventory tracking. Covers vendor scoping, shipment tracking, security setup, asset documentation, and production handoff — gaps that Snipe-IT doesn't natively handle.

## Structure

| Section | What it covers |
|---------|---------------|
| `vendor-scoping/` | Vendor evaluation, RFx process, relationship tracking |
| `shipment-tracking/` | Receiving log, carrier integration, discrepancy handling |
| `security-setup/` | Baseline hardening, encryption, compliance checks |
| `documentation/` | Asset documentation standards, runbooks, wiki integration |
| `production-handoff/` | Staging-to-production workflow, acceptance testing, go-live |

## Usage

Each section contains a README with process overview, plus templates (markdown, CSV) you can import into your ticketing system, wiki, or Snipe-IT notes fields.
